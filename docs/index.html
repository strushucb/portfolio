B.E.C.C.A.

Used natural language processing and machine learning techniques on Hillary Clinton’s State-released emails to extract important people, topics and tone and visualize them in a graph database (Neo4J). The database was built using Python and various NLP & ML libraries while the web application is written in Node.js using popoto.js and grapheneDB.

Bay Area Surveillance

Used visualization and HCI principles to inform Bay Area residents of the government surveillance technologies used in their area. This project combined user research, design, and front end development. I used D3 + Javascript to implement the face bubble and sankey diagrams.
